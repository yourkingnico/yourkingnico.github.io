Base Conversion Learning Tool

The Base Conversion Tool seen above was developed over a nine week period in my Advanced Programming class. I originally pitched this idea to my group as an overall learning tool that would allow the user to learn the main base conversions, practice at varying difficulties in a quiz mode, and yield a quick and agile base calculator. I was extremely satisfied that we accomplished these goals, although it was not easy. Some of my contributions to this project include: implementing the Observer Design Pattern, managing project dependencies with Apache Maven, developing both the quiz mode and calculator in javaFX, and building the executable JAR. This group project both challenged me and aided in further developing my Java skills in a collaborative environment. Please be sure to checkout some of my code snippets and contributions here.

But don't just take my word for it! Download it and give it a test drive (requires Java Runtime Environment).
